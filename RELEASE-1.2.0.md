# LangSpot 1.2.0 — Agenda mensal

## Novidades

- Alternância entre visualização semanal e mensal.
- Preferência de visualização salva no navegador.
- Navegação por semana ou mês, com botão **Hoje**.
- Grade mensal de 6 semanas, incluindo dias adjacentes.
- Até três aulas exibidas por dia e contador para aulas adicionais.
- Clique em uma aula para editar.
- Clique em um dia vazio ou no botão `+` para agendar com a data preenchida.
- Painel de detalhes para dias com muitas aulas.
- Layout responsivo para web, AppImage e Android.

## Publicação

```bash
npm install
npm run check
git add .
git commit -m "Release LangSpot 1.2.0 monthly calendar"
git push
```
