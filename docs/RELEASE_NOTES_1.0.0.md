# LangSpot 1.0.0

Primeira versão estável do LangSpot, uma plataforma para professores independentes de idiomas e seus alunos.

## Principais recursos

- Cadastro e acompanhamento de alunos
- Agenda e histórico de aulas
- Progresso por habilidades
- Materiais, links e PDFs privados
- Tarefas, entregas, feedback e notas
- Controle financeiro básico
- Relatórios e exportação para PDF
- Notificações internas
- Portal do aluno
- Perfis, avatares e primeiro acesso
- Layout responsivo e tema escuro
- Integração com Supabase e políticas RLS

## Validação

A versão foi validada com TypeScript e build de produção por meio de `npm run check`.
