# LangSpot 1.4.0 — Student Experience

## Novidades

- Dashboard do aluno redesenhado com próxima aula, tarefas pendentes, metas e materiais.
- Linha do tempo unificada com aulas, tarefas, metas concluídas e registros de estudo.
- Ações rápidas para diário, flashcards e metas.
- Área de metas com progresso, data-alvo, categorias e histórico de conquistas.
- Diário de aprendizagem com tempo de estudo, humor e novas palavras.
- Layout responsivo para web, AppImage e Android.

npm install
